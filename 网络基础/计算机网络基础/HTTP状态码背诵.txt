

1** 消息（临时响应） ：临时响应并要求请求者继续执行操作
100 Continue ：服务端已收到请求头并请客户端继续发送请求主体；
101 Switching Protocols : 服务端收到请求者的协议切换请求，并已确定后准备更换；

2** 成功 ： 表示成功处理了请求的状态码
200 Ok : 请求已成功，请求所希望的响应头或数据体将随此响应返回;
201 Created : 请求已经被实现，而且有一个新的资源已经依据请求的需要而创建;
202 Accepted : 请求已接受，但尚未处理；
203 Non-Authoritative Information ：请求已被服务器成功处理，但返回的信息可能来自另一个源；
204 No Content : 服务器成功处理请求但没有返回任何内容；
205 Reset Content : 服务器成功处理了请求，但没有返回任何内容。与204响应不同，此响应要求请求者重置文档视图
206 Partial Content : 服务器已经成功处理了部分GET请求

3**重定向 : 这类状态码代表需要客户端采取进一步的操作才能完成请求。
301 Moved Permanently : 被请求的资源已永久移到了新的位置，并且将来任何对此资源的引用都应该使用本响应返回的若干个URL之一
302 Found ： 要求客户端执行临时重定向（临时移动），但请求者应继续使用原有位置来进行以后的请求；

4**请求(客户端)错误 ：这些状态代码表示请求可能出错，妨碍了服务器的处理
400 Bad Request : 由于明显的客户端错误（例如，格式错误的请求语法，太大的大小，无效的请求消息或欺骗性路由请求），服务器不能或不会处理该请求
401  Unauthorized (未授权) ：请求要求身份验证。 对于需要登录的网页，服务器可能返回此响应
403 Forbidden ： 服务器已经理解请求，但是拒绝执行它。与401响应不同的是，身份验证并不能提供任何帮助，而且这个请求也不应该被重复提交
404 Not Found : 请求失败，请求所希望得到的资源未被在服务器上发现，但允许用户的后续请求
405 Method Not Allowed : 禁用请求中指定的方法来请求资源
406 Not Acceptable ：请求的资源的内容特性无法满足请求头中的条件，因而无法生成响应实体，该请求不可接受

5**服务器错误 ： 表示服务器无法完成明显有效的请求，这些错误可能是服务器本身的错误，而不是请求出错
500 Internal Server Error : 通用错误消息，服务器内部遇到了一个未曾预料的状况，导致了它无法完成对请求的处理。
501 Not Implemented ： 服务器不具备完成请求的功能
502 Bad Gateway ： 服务器作为网关或代理，从上游服务器收到无效响应
503 Service Unavailable ： 由于临时的服务器维护或者过载，服务器当前无法处理请求
504 Gateway Timeout ： 服务器作为网关或代理，但是没有及时从上游服务器收到请求
505 HTTP Version Not Supported ： 服务器不支持请求中所用的 HTTP 协议版本
