slice：	返回新数组或字符串，
		  str.slice(1,4)  //不包含尾项
		  返回指定下标的数组或字符串

splice：改变原有数组（只能用于数组，不能字符串）
			str.splice(1,4)  //不包含尾项
			返回指定下标范围的数组
			
split：	返回新数组，将字符串转化数组
		  str.split( )  //转化为一个数组 [ 'djksjdl' ]
		  str.split('') //将每个字符都分隔成一个数组项[  'd', 'j', 'k',  's', 'j', 'd','l']
		  
join：	返回新数组，将数组合并成为字符串
		str.join( ) // 2,9,3,8,9,2
		 str.join('') // 293892 
		 str.join('|') // 2|9|3|8|9|2
		 插入的符号为合并后的分隔符
